# ParkNear

Приложение для поиска и управления парковками.

## Установка

1. Клонируйте репозиторий.
2. Откройте `index.html` в браузере (или используйте live-server).

## Настройка

- API-ключ Яндекс.Карт уже встроен.
- Firebase настроен для демонстрационных целей.

## Структура

- `css/` – стили
- `js/` – модули JavaScript
  - `config/` – конфигурация
  - `models/` – работа с данными
  - `services/` – внешние сервисы
  - `controllers/` – логика приложения
  - `views/` – рендеринг UI
  - `components/` – переиспользуемые элементы
  - `utils/` – утилиты
